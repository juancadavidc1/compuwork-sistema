# Sistema de Gestión RRHH para Compuwork

Este proyecto implementa un sistema para la gestión de empleados,departamentos y reportes de desempeño para la empresa.

## Funcionalidades

- Crear empleados
- Manejar empleados permanentes y temporales
- Crear departamentos
- Asignar empleados a departamentos
- Generar reportes de desempeño
